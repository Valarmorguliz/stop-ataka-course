# Платформа онлайн-курсов

Небольшая платформа для размещения онлайн-курсов с иерархической структурой, домашними заданиями и уведомлениями.

## Функционал

- Регистрация и аутентификация пользователей (студенты и учителя)
- Иерархия курсов: Курс -> Блоки -> Уроки
- Уроки содержат текст, картинки, видео (iframe)
- Домашние задания с возможностью сдачи и проверки
- Переключение светлой/темной темы
- Локальное хранение данных с шифрованием

## Установка и запуск локально

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите сервер: `npm run dev`

## Развертывание на GitHub Pages

1. Соберите проект: `npm run build`
2. Разверните папку `.next/static` или используйте GitHub Actions для автоматического развертывания.
3. Для SPA поддержки добавьте 404.html в public.

## Использование

- Зарегистрируйтесь как студент или учитель
- Учителя могут создавать курсы, блоки и уроки
- Студенты могут просматривать уроки и сдавать задания
- Данные хранятся локально в браузере, зашифрованные

## Структура контента урока

Контент урока хранится как JSON:

```json
{
  "text": "Текст урока",
  "images": ["url1", "url2"],
  "video": "https://example.com/embed"
}
```

Видео вставляется как iframe с настраиваемым размером.
